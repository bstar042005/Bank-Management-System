# Bank Management System (C++)

A console-based Bank Management System developed using Object-Oriented Programming (OOP) concepts in C++.

## Features

- Add Customer
- Create Savings Account
- Create Current Account
- Deposit Money
- Withdraw Money
- Transfer Money
- Display Customer Details
- Display Account Details
- File Handling for Data Storage

## Concepts Used

- Classes & Objects
- Inheritance
- Polymorphism
- Encapsulation
- Abstraction
- Virtual Functions
- File Handling
- STL Vector

## Technologies

- C++
- Object-Oriented Programming
- File I/O

## Compile

```bash
g++ *.cpp -std=c++17 -o bank.exe
